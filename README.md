
# Cinemore Player

https://cinemore.com.cn/server

A high-performance, cross-platform video player application for personal media library management and playback, designed to be the best companion for NAS users and provide an elegant user experience.

Integrate users 'video resources scattered in local, NAS, and cloud storage, form a unified private library through intelligent scraping and automatic classification, and provide a consistent high-quality, low-power playback experience across the platform, and provide offline caching function to facilitate users. Watch in a network-free environment.
